Chapter 7: Statistical Intervals Based on a Single Sample

Introduction (Page 276)

In the previous chapter, we learned about point estimates, which are single numbers used to estimate a population parameter (like μ or p). For example, we might use the sample mean x = 9322.7 g as a point estimate for the true average breaking strength μ of paper towels.

However, a major limitation of a point estimate is that it doesn't tell us anything about how precise or reliable it is. Due to sampling variability (different samples give different results), it's almost never the case that our estimate (x) is exactly equal to the true parameter value (μ). The point estimate alone doesn't indicate how close it might be to the true value.

An alternative approach is to calculate an interval estimate, more commonly called a confidence interval (CI). This is an entire interval of plausible values for the parameter.

  • Confidence Level: To calculate a CI, we first choose a confidence level, which represents the degree of reliability of the interval. Common choices are 95%, 99%, and 90%.
  • Interpretation: For instance, a 95% CI for the true average breaking strength μ might be (9162.5, 9482.9). This means we are 95% confident that the true value of μ lies between 9162.5 and 9482.9. A 95% confidence level implies that if we were to repeat the sampling process many times and calculate a CI for each sample, about 95% of those intervals would successfully capture the true value of the parameter (μ in this case), while 5% would not.
  • Precision and Width: The width of the confidence interval gives information about the precision of our estimate.
    • A narrow interval (at a high confidence level) suggests that we have a precise estimate of the parameter. (See Brand 1 in Figure 7.1, page 277).
    • A wide interval suggests a lot of uncertainty about the true value of the parameter. (See Brand 2 in Figure 7.1, page 277).

The higher the confidence level we choose, the more strongly we believe the true parameter value lies within the interval. However, as we'll see, higher confidence levels usually lead to wider intervals.

Section 7.1: Basic Properties of Confidence Intervals (Page 277)

Let's start with a simple, though somewhat unrealistic, situation to understand the basic concepts. We want to estimate the population mean μ, and we make two assumptions:

  1. The population distribution is normal.
  2. The population standard deviation σ is known.

(Note: Assuming σ is known when μ is unknown is often unrealistic, but it simplifies the initial explanation. We'll relax this assumption later.)

Deriving the Confidence Interval (Normal Population, σ known)

  • Starting Point: If we take a random sample X1, ..., Xn from a normal population with mean μ and known standard deviation σ, we know the sample mean X is also normally distributed with mean E(X) = μ and standard deviation σX = σ/√n.
  • Standardizing: We can standardize X to get a standard normal variable Z:
    Z = X − μ σ/√n   (Equation 7.1, page 277)
  • Probability Statement: Since Z follows a standard normal distribution, we know P(−1.96 < Z < 1.96) = 0.95. Substituting the expression for Z:
    P( −1.96 < X − μ σ/√n < 1.96 ) = 0.95   (Equation 7.2, page 278)
  • Isolating μ: Algebraically manipulate the inequalities to isolate μ:
    X − 1.96 σ √n < μ < X + 1.96 σ √n
  • The Random Interval: The probability statement becomes:
    P( X − 1.96 σ √n < μ < X + 1.96 σ √n ) = 0.95   (Equation 7.3, page 278)
    The interval
    ( X − 1.96 σ√n , X + 1.96 σ√n )
    (Equation 7.4, page 278) is a random interval. Its location depends on X, but its width is fixed. Equation 7.3 means there's a 95% probability this interval contains μ before sampling.

Calculating the Confidence Interval

  • Definition (Page 279): After collecting data (x1, ..., xn) and calculating x, substitute x for X. The resulting fixed interval is a 95% confidence interval for μ.
  • Formula: 95% CI for μ (Normal pop, σ known):
    ( x − 1.96 σ√n , x + 1.96 σ√n )
    or concisely:
    x ± 1.96 σ √n

Example 7.1 / 7.2: Keyboard Height (Pages 277, 279)

  • Goal: Estimate μ, true average preferred keyboard height.
  • Assumptions: Normal population, σ = 2.0 cm (known).
  • Data: n = 31, x = 80.0 cm.
  • Calculation: 95% CI is 80.0 ± 1.96 * (2.0 / √31) = 80.0 ± 0.704 ≈ (79.3, 80.7) cm.
  • Conclusion: We are 95% confident that the true average preferred keyboard height μ lies between 79.3 cm and 80.7 cm.

Interpreting a Confidence Level (Page 279)

This is crucial. Once a specific interval like (79.3, 80.7) is calculated, it's incorrect to say "the probability that μ is between 79.3 and 80.7 is 0.95." Why? Because μ is fixed (but unknown) and the interval is fixed. μ is either in the interval or not.

The correct interpretation relates to the long-run performance of the method:

  • If we were to repeat the process (take sample, compute 95% CI) many times, about 95% of the intervals constructed would capture the true value of μ, and about 5% would miss it. (See Figure 7.3, page 280).
  • Our confidence is in the procedure. We don't know if our particular interval is one of the "good" 95% or "bad" 5%, but the method works 95% of the time.

Other Levels of Confidence (Page 280)

  • To change the confidence level, we change the z critical value.
  • Recall: zα is the value with area α to its right.
  • For a 100(1 - α)% CI, we need the middle 1 - α area, leaving α/2 in each tail. The critical value is zα/2. (See Figure 7.4, page 281).
  • Common values:
    • 95% CI: α=0.05, zα/2 = z.025 = 1.96.
    • 99% CI: α=0.01, zα/2 = z.005 ≈ 2.58.
    • 90% CI: α=0.10, zα/2 = z.05 = 1.645.
  • Definition (Page 281): A 100(1 - α)% CI for μ (Normal population, σ known) is:
    ( x − zα/2 σ√n , x + zα/2 σ√n )   (Equation 7.5)
    or concisely:
    x ± zα/2 σ √n
    This is: (Point estimate) ± (z critical value) * (standard error).

Example 7.3: Housing Units (Page 281)

  • Goal: Estimate μ, true average hole diameter.
  • Assumptions: Normal distribution, σ = 0.100 mm (known).
  • Data: n = 40, x = 5.426 mm.
  • Desired Confidence Level: 90%. α = 0.10, α/2 = 0.05, z.05 = 1.645.
  • Calculation: 90% CI is 5.426 ± 1.645 * (0.100 / √40) = 5.426 ± 0.026 = (5.400, 5.452) mm.
  • Conclusion: We are 90% confident the true average hole diameter is between 5.400 mm and 5.452 mm.

Confidence Level, Precision, and Sample Size (Page 281)

  • Confidence vs. Width: For fixed n and σ, a higher confidence level requires a larger zα/2, resulting in a wider (less precise) interval.
    • Width (95%) = 2 * 1.96 * (σ/√n)
    • Width (99%) = 2 * 2.58 * (σ/√n)
  • Sample Size vs. Width: For fixed confidence level and σ, increasing n decreases the width (more precise).

Determining Sample Size (Page 282)

We can find the required sample size n for a desired interval width w, if σ is known.

  • Width w = 2 * zα/2 * (σ/√n).
  • Solving for n:
    n = [ 2 zα/2 σ w ]2
  • Alternatively, specify the desired bound on the error of estimation (B), which is half the width (B = w/2). Then B = zα/2 * (σ/√n).
  • Solving for n using B:
    n = [ zα/2 σ B ]2
  • Note: Always round the calculated n up to the next whole number.

Example 7.4: Sample Size for Response Time (Page 282)

  • Goal: Estimate μ, true average response time.
  • Assumptions: Normal distribution, σ = 25 msec (known).
  • Desired Confidence Level: 95% (z.025 = 1.96).
  • Desired Width: w = 10 msec (at most).
  • Calculation: n = [ 2 * 1.96 * 25 / 10 ]² = [9.8]² = 96.04.
  • Result: Need sample size n = 97.

General Method for Deriving a Confidence Interval (Pages 282-283)

The process used can be generalized to find a CI for a parameter θ.

  1. Find a random variable h(X1, ..., Xn; θ) such that:
    • It depends on the sample and θ.
    • Its probability distribution does not depend on θ or other unknowns (it's a pivotal quantity).
    Example: h = (X - μ) / (σ/√n) has a N(0,1) distribution.
  2. Find constants 'a' and 'b' such that for confidence level 1 - α:
    P( a < h(X1, ..., Xn; θ) < b ) = 1 − α   (Equation 7.6)
    Example: For 95%, a = −z.025 and b = z.025.
  3. Algebraically manipulate a < h(...) < b to isolate θ:
    P( l(X1, ..., Xn) < θ < u(X1, ..., Xn) ) = 1 − α
  4. The CI is (l(x1, ..., xn), u(x1, ..., xn)).

Example 7.5: CI for Exponential Parameter λ (Page 283)

  • Goal: 95% CI for rate λ (where mean μ = 1/λ).
  • Data: n = 10, Σxi = 550.87.
  • Pivotal Quantity: h = 2λΣXi follows a chi-squared (χ²) distribution with ν = 2n degrees of freedom (df).
  • Critical Values: For 95% CI and df = 2n = 20, we need χ² values cutting off 2.5% in each tail (from Table A.7):
    • Lower: χ²0.975,20 = 9.591
    • Upper: χ²0.025,20 = 34.170
  • Probability Statement: P( 9.591 < 2λΣXi < 34.170 ) = 0.95.
  • Isolating λ: Divide by 2ΣXi.
    P( χ21−α/2, 2n 2ΣXi < λ < χ2α/2, 2n 2ΣXi ) = 1 − α
  • Calculating the 95% CI for λ: Substitute Σxi = 550.87.
    • Lower limit = 9.591 / (2 * 550.87) = 0.00871
    • Upper limit = 34.170 / (2 * 550.87) = 0.03101
    The 95% CI for λ is (0.00871, 0.03101).
  • CI for Mean Lifetime (μ = 1/λ): Take reciprocal of limits and reverse order.
    • Lower limit for μ = 1 / 0.03101 = 32.24
    • Upper limit for μ = 1 / 0.00871 = 114.87
    The 95% CI for μ is (32.24, 114.87) minutes. Very wide.
  • Note: Limits are not symmetric around the point estimate.

Bootstrap Confidence Intervals (Page 284)

Bootstrap methods provide an alternative way to construct CIs, especially when the theory is complex. One method uses percentiles of the bootstrap distribution of (θ* − θ):

  1. Obtain B bootstrap samples, calculate estimate θ* for each. Calculate original estimate θ.
  2. Compute B differences: di = θi* − θ.
  3. Find the (α/2)th and (1−α/2)th percentiles of these differences (call them dlow and dhigh).
  4. The 100(1−α)% bootstrap CI is:
    ( θ − dhigh , θ − dlow )   (Based on Formula 7.7)

Example Reference: This structure matches the standard CI for the mean when using the appropriate percentiles of X - μ.

Section 7.2: Large-Sample Confidence Intervals for a Population Mean and Proportion

Here, we relax the assumptions from Section 7.1.

A Large-Sample Interval for μ (Page 286)

  • Motivation: Population might not be normal, or σ is unknown, but sample size n is large.
  • Central Limit Theorem (CLT): If n is large (e.g., n > 30 or n > 40), then X is approximately normal, and Z = (X - μ) / (σ/√n) is approximately standard normal.
  • Problem: Still requires unknown σ.
  • Solution for Large n: Replace σ with the sample standard deviation S. For large n, S is close to σ. The statistic
    Z ≈ X − μ S/√n
    is approximately standard normal.
  • Proposition (Page 286): If n is sufficiently large (n > 40 recommended), an approximate 100(1 - α)% CI for μ is:
    ( x − zα/2 S√n , x + zα/2 S√n )   (Equation 7.8)
    or concisely:
    x ± zα/2 S √n
  • Interpretation: Point estimate ± (z critical value) * (estimated standard error). Here, sx = S/√n.

Example 7.6: Porsche Odometer Readings (Pages 286-287)

  • Goal: Estimate μ, true average odometer reading.
  • Data: n = 50 (> 40). x = 45,679.4 miles. s = 26,641.675 miles.
  • Desired Confidence Level: Approx. 95% (z.025 = 1.96).
  • Calculation: 45,679.4 ± 1.96 * (26,641.675 / √50) = 45,679.4 ± 7384.7 = (38294.7, 53064.1) miles.
  • Conclusion: Approx. 95% confident the true average is between 38,294.7 and 53,064.1 miles. Wide due to large s.

Sample Size Determination (Large n, unknown σ) (Pages 287-288)

Using n = [ 2 * zα/2 * σ / w ]² requires σ.

  • Approach: Estimate σ before collecting data.
    • Use σ from a previous study.
    • Conduct a pilot study to get s.
    • Estimate Range (Max - Min) and use σ ≈ Range / 4 (or Range / 6).
  • Use the estimated σ in the sample size formula. Round n up.

Example 7.7: Sample Size for Charge-to-Tap Time (Page 288)

  • Goal: Estimate μ.
  • Desired Level: 95% (z.025 = 1.96).
  • Desired Precision: Bound B = 5 minutes.
  • Estimating σ: Range ≈ 440 - 320 = 120 min. Estimate σ ≈ 120 / 4 = 30 min.
  • Calculation: n = [ zα/2 * σ / B ]² = [ 1.96 * 30 / 5 ]² = [11.76]² = 138.3.
  • Result: Use n = 139.

General Large-Sample Confidence Interval Structure (Page 288)

If an estimator θ for θ is approximately normal, unbiased, and has an estimated standard error sθ, then an approximate 100(1 - α)% CI for θ is:

θ ± zα/2 sθ

(Point estimate ± (z critical value) * (estimated standard error))

A Confidence Interval for a Population Proportion p (Page 289)

  • Parameter: p = population proportion of successes.
  • Estimator: p = X/n (sample proportion).
  • Properties (Large n): p is approx. normal, unbiased, standard error σp = √[p(1-p)/n].
  • Score Confidence Interval (Wilson) (Page 289): Derived by solving Z² = z²α/2 for p, where Z = (p - p) / √[p(1-p)/n]. The resulting approximate 100(1 - α)% CI limits for p are:
    p + z2α/22n ± zα/2 √[ p(1−p)n + z2α/24n2 ] 1 + z2α/2n
    (This structure corresponds to the Proposition box 7.10 and the formula above it).
  • Traditional (Wald) Interval (Formula 7.11, Page 289): Simpler approximation valid for very large n:
    p ± zα/2 √[ p(1−p) n ]
    This fits the general form θ ± zα/2 * sθ with estimated standard error sp = √[p(1-p)/n].
  • Why Use the Score Interval? (Page 290)
    • Its actual confidence level is usually closer to the nominal level (e.g., 95%) than the Wald interval, especially if p is near 0 or 1, or if n is not huge. (See Figure 7.6).
    • It performs well even if conditions like np≥10 and n(1-p)≥10 fail.
    • Recommendation: Use the Score (Wilson) interval generally.

Example 7.8: Substrate Ignition (Pages 290-291)

  • Goal: Estimate p, true proportion of ignitions.
  • Data: n = 48, X = 16. p = 16/48 ≈ 0.333.
  • Desired Level: Approx. 95% (z.025 = 1.96).
  • Calculation (Score Interval): Using the formula above gives (0.216, 0.474). (Intermediate steps involve calculating the adjusted center p̃ ≈ 0.345 and the margin of error ≈ 0.129).
  • Calculation (Traditional/Wald Interval): 0.333 ± 1.96 * √[(0.333)(0.667)/48] = 0.333 ± 0.133 = (0.200, 0.466).
  • Intervals are similar here.

Sample Size for Estimating p (Page 291)

  • To get a desired width w for the CI for p.
  • Approximate formula based on Wald width:
    n ≈ 4 z2α/2 p(1−p) w2
  • Problem: Requires p before sampling.
  • Solutions:
    • Conservative: Use p = 0.5 (maximizes p(1-p) = 0.25). Formula becomes:
      n ≈ z2α/2 w2
      (Because 4 * 0.25 = 1). Guarantees width ≤ w.
    • Educated Guess: Use prior information to guess p (use value closest to 0.5).
  • Always round n up.

Example 7.9: Sample Size for Ignition Proportion (Page 291)

  • Goal: n for 95% CI width w = 0.10.
  • Level: 95% (z.025 = 1.96). Width w = 0.10.
  • Approach: Conservative (p=0.5).
  • Calculation (Approx): n ≈ (1.96)² / (0.10)² = 3.8416 / 0.01 = 384.16.
  • Result: Need n = 385. (Text uses exact formula giving n=381).

One-Sided Confidence Intervals (Confidence Bounds) (Pages 291-292)

  • Used for an upper or lower bound only.
  • Concept: Capture 100(1-α)% area in one tail. Use critical value zα instead of zα/2.
  • Large-Sample Bounds for μ (Page 292):
    • Upper bound: μ < x + zα * (S/√n)
    • Lower bound: μ > x − zα * (S/√n)
  • Bounds for p: Use zα in the Score CI formula and take only the relevant limit.

Example 7.10: Titanium Roughness (Page 292)

  • Goal: 95% upper confidence bound for μ.
  • Data: n = 54, x = 1.9042, s = 0.1455.
  • Level: 95% (one-sided), α = 0.05. Need zα = z.05 = 1.645.
  • Calculation: μ < 1.9042 + 1.645 * (0.1455 / √54) = 1.9042 + 0.0326 = 1.9368.
  • Conclusion: Approx. 95% confident that μ < 1.9368 μg.

Section 7.3: Intervals Based on a Normal Population Distribution (Page 295)

This section applies when:

  • Sample size n is small (CLT may not apply).
  • Assume the population distribution is normal.
  • Population standard deviation σ is unknown.
  • Motivation: When n is small, replacing σ with S introduces significant extra variability. (X - μ) / (S/√n) is no longer standard normal.
  • Assumption: Population is normal, N(μ, σ).
  • The T statistic: Consider the statistic:
    T = X − μ S/√n   (Equation 7.13)
  • Theorem (Page 295): When the population is normal, T follows a t distribution with n − 1 degrees of freedom (df).

Properties of t Distributions (Page 296)

  • Family of distributions specified by degrees of freedom (ν = df).
  • Shape: Bell-shaped, centered at 0 (like z).
  • Spread: More spread out (heavier tails) than the standard normal (z) curve. Reflects extra uncertainty from using S.
  • Effect of df: As ν increases, the t curve gets closer to the z curve (less spread, lighter tails).
  • Limit: As ν → ∞, the t curve becomes the z curve.
  • (See Figure 7.7, page 296).
  • Why df = n - 1? S depends on Xi - X, which sum to 0. Only n-1 are independent.

t Critical Values (Page 296)

  • Notation: tα,ν = value on t curve with ν df such that area to its right is α. (See Figure 7.8, page 297).
  • By symmetry, -tα,ν cuts off area α in the left tail.
  • Use Appendix Table A.5 (columns for α, rows for ν).
  • Example: t.05,15 = 1.753. t.01,22 = 2.508.
  • For a 100(1 - α)% CI, we need the value tα/2,ν cutting off area α/2 in the upper tail.

The One-Sample t Confidence Interval (Page 297)

  • Derivation: Start with P( -tα/2,n-1 < T < tα/2,n-1 ) = 1 - α. Isolate μ.
  • Proposition (Page 297): For a random sample from a normal population, a 100(1 - α)% CI for μ is:
    ( x − tα/2, n−1 S√n , x + tα/2, n−1 S√n )   (Equation 7.15)
    or concisely:
    x ± tα/2, n−1 S √n
  • One-Sided t Bounds (Page 297): Use tα,n-1 critical value.
    • Upper bound: μ < x + tα,n-1 * (S/√n)
    • Lower bound: μ > x − tα,n-1 * (S/√n)

Example 7.11: Sweetgum Lumber MOR (Pages 297-298)

  • Goal: Estimate μ, true average MOR.
  • Assumption: MOR population is approx. normal (Figure 7.9). σ unknown. n=30 (can use t-interval).
  • Data: n = 30, x = 7203.191 psi, s = 543.5400 psi.
  • Desired Level: 95% CI. α=0.05, α/2=0.025. df = n-1 = 29. Critical value t.025,29 = 2.045 (Table A.5).
  • Calculation: 7203.191 ± 2.045 * (543.5400 / √30) = 7203.191 ± 202.938 = (7000.253, 7406.129) psi.
  • Conclusion: We are 95% confident that the true average MOR is between 7000.253 psi and 7406.129 psi.
  • One-Sided Example: 95% lower bound uses t.05,29 = 1.699. Bound is 7203.191 - 1.699 * (543.54 / √30) ≈ 7034.59 psi. 95% confident μ > 7034.59 psi.

Controlling Interval Width (Page 299)

Selecting n for a desired width using the t-interval is difficult because the width 2 * tα/2,n-1 * (S/√n) depends on S (unknown) and the critical value t itself depends on n. Requires iteration or other methods.

Robustness of the t-Interval (Page 301)

  • The t-interval is robust to moderate departures from the normality assumption, especially if n is not too small.
  • If the population is slightly non-normal, the actual confidence level will be close to the nominal level (e.g., 95%).
  • However, if n is small and the population is highly non-normal (skewed, outliers), the t-interval may not be reliable. Consider bootstrap or non-parametric methods.
