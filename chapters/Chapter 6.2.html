Section 6.2: Methods of Point Estimation

(Summary based on pages 264-272)

This section introduces two specific methods for deriving point estimators for unknown parameters:

  • The Method of Moments (MoM)
  • The Method of Maximum Likelihood (MLE)

These are called "constructive" methods because they provide a clear procedure for finding an estimator in a given situation.

Comparison: While MLEs often have better efficiency properties (meaning they tend to have lower variance, especially for large samples), they can sometimes be more computationally intensive than MoM estimators. Both methods can sometimes yield unbiased estimators.

The Method of Moments (MoM) (Pages 264-265)

Basic Idea: Match sample characteristics (specifically, sample moments) to their corresponding theoretical population moments, which are functions of the unknown parameters. Then, solve the resulting equations for the parameters.

Definition: Population and Sample Moments (Page 264)

For a random sample X1, ..., Xn from a pmf or pdf f(x):

  • The kth population moment is E(Xk). This is a characteristic of the distribution.
  • The kth sample moment is:
    1 n n Σ i=1 Xik
    (the average of the sample values raised to the power k).
  • The first population moment (k=1) is E(X) = μ.
  • The first sample moment (k=1) is:
    X = 1 n n Σ i=1 Xi
    (the sample mean).
  • The second population moment (k=2) is E(X2).
  • The second sample moment (k=2) is:
    1 n n Σ i=1 Xi2

Definition: Method of Moments Estimators (Page 265)

  • Let the distribution depend on m unknown parameters: θ1, θ2, ..., θm.
  • To find the MoM estimators, set the first m sample moments equal to the first m population moments:
E(X) = 1 n n Σ i=1 Xi = X
E(X2) = 1 n n Σ i=1 Xi2

...

E(Xm) = 1 n n Σ i=1 Xim
  • The population moments E(Xk) will be functions of θ1, ..., θm.
  • Solve this system of m equations for the m unknown parameters θ1, ..., θm. The solutions are the MoM estimators θ1, ..., θm.

Example 6.12: Exponential Distribution (Page 265)

  • Distribution: Exponential with one parameter, λ.
  • We need one equation (m=1). Match the first moments: E(X) = X.
  • For exponential, E(X) = 1/λ.
  • Equation: 1/λ = X.
  • Solving for λ gives the MoM estimator:
    λ = 1 X

Example 6.13: Gamma Distribution (Page 265)

  • Distribution: Gamma with two parameters, α and β.
  • We need two equations (m=2). Match the first two moments:
    • E(X) = X
    E(X2) = 1 n n Σ i=1 Xi2
  • For Gamma, E(X) = αβ and E(X²) = V(X) + [E(X)]² = αβ² + (αβ)² = α(α+1)β².
  • Equations:
    α β = X    (1)
    α(α+1)β2 = 1 n n Σ i=1 Xi2    (2)
  • Solving this system leads to the MoM estimators:
    β = 1n n Σ i=1 Xi2 − X2 X
    α = X β = X2 1n n Σ i=1 Xi2 − X2
  • Illustration: Using survival time data (n=20, x = 113.5, (1/n)Σxi² = 14087.8): α ≈ 10.7, β ≈ 10.6.

Example 6.14: Generalized Negative Binomial Distribution (Page 266)

  • Distribution: Depends on parameters r and p. Need two equations (m=2).
  • Match first two moments: E(X) = X, E(X²) = (1/n)ΣXi².
  • For Neg Binomial, E(X) = r(1−p)/p and E(X²) = V(X) + [E(X)]².
  • Solving the system yields MoM estimators:
    p = X 1n n Σ i=1 Xi2 − X2
    r = X2 1n n Σ i=1 Xi2 − X2 − X
  • Illustration: Hockey goals data (n=420 games, x = 2.98, (1/n)Σxi² = 12.40): p ≈ 0.85, r ≈ 16.5.
  • Issue: The denominator for r can be negative if the sample variance is less than the sample mean. Since r must be positive, a negative estimate suggests the model might be inappropriate or the MoM estimator is flawed here.

Maximum Likelihood Estimation (MLE) (Pages 266-268)

Introduction: Developed by R.A. Fisher in the 1920s. Often recommended, especially for large samples, due to good efficiency properties.

Basic Idea: Find the parameter value(s) that make the observed sample data most probable or most likely to have occurred.

Example 6.15: Bernoulli Trials (Strong Passwords) (Pages 266-267)

  • Scenario: n=10 individuals. Let Xi = 1 if the ith person has strong protection, Xi = 0 otherwise. Assume independence. p = P(strong protection).
  • Data (Σxi = 3, n=10).
  • Probability of the observed sample (joint pmf, likelihood function):
    L(p) = f(x1, ..., x10; p) = pΣi=1nxi (1−p)n − Σi=1nxi = p3 (1−p)7
  • Question: For which value of p is this probability maximized?
  • Log-Likelihood: It's often easier to maximize the natural logarithm:
    ln[L(p)] = ln[p3(1−p)7] = 3ln(p) + 7ln(1−p)
  • Calculus: Find the derivative with respect to p and set it to zero:
    d dp [ln(L)] = 3p − 71−p
  • Set to 0: 3/p = 7/(1−p) => p = 3/10 = 0.30.
  • Result: The value p=0.30 maximizes the likelihood. This is the Maximum Likelihood Estimate (MLE).
  • Note: If only X=3 (number of successes) was known, the pmf is Binomial. Maximizing this gives the same result.

Definition: Likelihood Function and MLE (Page 268)

  • Let X1, ..., Xn have a joint pmf or pdf f(x1, ..., xn; θ1, ..., θm).
  • When viewed as a function of the parameters θ1, ..., θm after observing data x1, ..., xn, it's the likelihood function:
    L(θ1, ..., θm) = f(x1, ..., xn; θ1, ..., θm)
  • The Maximum Likelihood Estimates (MLEs) θ1, ..., θm are the parameter values that maximize L.
  • The corresponding Maximum Likelihood Estimators replace xi with Xi.
  • Interpretation: The likelihood function measures how "likely" the observed sample is for different parameter values. MLEs make the observed data seem most plausible.

Example 6.16: Exponential Distribution (Page 268)

  • Sample: X1, ..., Xn from Exponential(λ).
  • Likelihood function:
    L(λ) = n Π i=1 (λe−λxi) = λn e−λΣi=1nxi
  • Log-Likelihood:
    ln[L(λ)] = n ln(λ) − λ n Σ i=1 xi
  • Derivative w.r.t. λ:
    d dλ [ln(L)] = nλ − n Σ i=1 xi
  • Set to 0: n/λ = Σxi => λ = n / Σxi = 1 / (Σxi/n) = 1/x.
  • MLE for λ:
    λ = 1 X
    (Same as MoM estimator, but not unbiased).

Example 6.17: Normal Distribution (Pages 268-269)

  • Sample: X1, ..., Xn from Normal(μ, σ²). Parameters are μ and σ².
  • Log-Likelihood:
    ln[L(μ, σ2)] = −n2ln(2π) − n2ln(σ2) − 12σ2 n Σ i=1 (xi−μ)2
  • Partial derivatives set to 0 yield:
    ∂∂μ [ln(L)] = 0  ⟹  n Σ i=1 (xi−μ)=0  ⟹  μ = x
    ∂∂(σ2) [ln(L)] = 0  ⟹  σ2 = 1n n Σ i=1 (xi−μ)2
  • Substituting μ=x into the second result gives the MLEs:
    μ = X
    σ2 = 1 n n Σ i=1 (Xi − X)2
  • Note: The MLE for σ² uses divisor n, unlike the unbiased estimator S² which uses n−1.

Example 6.18: Poisson Process in 2D (Page 269)

  • Scenario 1: Count Xi in n fixed regions Ri with areas a(Ri). Xi ~ Poisson(λa(Ri)). Parameter is λ (events/area).
  • Taking log-likelihood, differentiating w.r.t λ, and setting to 0 yields:
    λ = n Σ i=1 Xi n Σ i=1 a(Ri) = Total events observed Total area sampled
    (This MLE is unbiased).
  • Scenario 2: Distance sampling. Measure Yi = distance to nearest plant for n points. pdf of Y is fY(y; λ) = 2πλye−λπy², y≥0.
  • Finding the MLE yields:
    λ = n π n Σ i=1 Yi2
  • Comparison: The two methods give different estimators; choice depends on context (sparse vs. dense).

Example 6.19: Weibull Distribution (Page 270)

  • Sample: X1, ..., Xn from Weibull(α, β).
  • Taking partial derivatives of the log-likelihood w.r.t. α and β and setting to zero yields complex equations that usually require numerical methods to solve for the MLEs α and β.

Estimating Functions of Parameters (Page 270)

Proposition: The Invariance Principle

If θ1, ..., θm are the MLEs of θ1, ..., θm, then the MLE of any function h(θ1, ..., θm) is simply the function evaluated at the MLEs: h(θ1, ..., θm).

Example 6.20: Normal Standard Deviation (Page 270)

  • We found MLEs μ = X and σ² = Σ(Xi − X)² / n.
  • The standard deviation is σ = h(μ, σ²) = √(σ²).
  • By the invariance principle, the MLE of σ is:
    σ = √( σ2 ) = √[ 1 n n Σ i=1 (Xi − X)2 ]
  • Note: This differs from the sample standard deviation S.

Example 6.21: Weibull Mean (Pages 270-271)

  • The mean of a Weibull distribution is:
    μ = β Γ(1 + 1/α)
  • If α and β are the (numerically computed) MLEs of α and β.
  • The MLE of the mean μ is:
    μ = β Γ(1 + 1/α)
  • The text notes that X is unbiased for μ, but μ is the MLE and considered better for large n.
  • Illustration: Data gives α = 11.9731, β = 77.0153, yielding μ = 73.80 (close to x = 73.88).

Large Sample Behavior of the MLE (Page 271)

MLEs are popular largely because of their excellent properties when the sample size (n) is large.

Proposition: Under general conditions:

  • For large n, the MLE θ is approximately unbiased:
    E(θ) ≈ θ
  • For large n, the variance of the MLE, V(θ), is nearly the smallest possible variance among all estimators.

Conclusion: For large n, the MLE is approximately the MVUE (Minimum Variance Unbiased Estimator). Because of these properties and the availability of methods to find them, MLE is the most widely used estimation technique (requires specifying the distribution family).

Some Complications (Pages 271-272)

Calculus (setting derivatives to zero) doesn't always work to find the maximum of the likelihood function.

Example 6.22: Uniform Distribution U[0, θ] (Page 271)

  • Sample: X1, ..., Xn from Uniform[0, θ]. Parameter is θ.
  • Likelihood function:
    L(θ) = 1 θn    for 0 ≤ xi ≤ θ for all i
    Equivalently, L(θ) = 1/θn if θ ≥ max(x1, ..., xn), and L(θ) = 0 otherwise.
  • This function L(θ) is maximized when θ is as small as possible, subject to the constraint θ ≥ max(x).
  • The smallest possible value is θ = max(x1, ..., xn).
  • MLE for θ:
    θ = max(X1, ..., Xn)
  • Calculus failed because the maximum occurred at a boundary. This MLE is biased.

Example 6.23: Capture-Recapture (Page 272)

  • Scenario: Capture M animals, tag, release. Recapture n, find X tagged. Estimate total population N.
  • Parameter: N (integer ≥ M, ≥ n).
  • Distribution of X: Hypergeometric h(x; n, M, N).
  • Likelihood:
    L(N) = p(x; N) = (Mx) (N-Mn-x) (Nn)
  • Goal: Find integer N that maximizes L(N). Calculus is hard (discrete N).
  • Alternative: Look at the ratio L(N) / L(N−1).
    L(N) L(N−1) = (N−M)(N−n) N(N−M−n+x)
  • The likelihood increases (ratio > 1) as long as N < Mn/x.
  • Conclusion: L(N) increases until N reaches Mn/x, then decreases. The maximizing N is the largest integer ≤ Mn/x.
  • MLE for N:
    N = ⌊ Mn x ⌋
  • Illustration: M=200, n=100, x=11 => N = floor(200*100/11) = floor(1818.18) = 1818.
  • Intuition: Equates sample proportion tagged (x/n) to population proportion tagged (M/N).

M-Estimators (Page 272)

Mentioned briefly as a robust estimation technique related to MLE. Instead of maximizing n Σ i=1 ln[f(xi; θ)] , M-estimators maximize a different objective function n Σ i=1 ρ(xi; θ) , where ρ is chosen for robustness properties, especially when the exact form of f is uncertain but symmetry is assumed.
