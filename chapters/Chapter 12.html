Chapter 12: Simple Linear Regression and Correlation

Introduction (Page 487)

Previously, we compared parameters between different populations. Regression analysis studies the relationship between two (or more) variables, aiming to predict one (y) from the other (x).

  • Deterministic Relationships: y is completely determined by x.
    • Example: Cost y = 25 + 0.3x (miles).
    • Example: Distance y = v0t + (1/2)at².
  • Nondeterministic (Probabilistic) Relationships: Knowing x doesn't perfectly determine y, but helps predict it. There's scatter.
    • Example: High school GPA (x) vs. College GPA (y).
    • Other examples: Age vs. vocabulary; engine size vs. fuel efficiency.

This chapter focuses on modeling linear probabilistic relationships.

12.1 The Simple Linear Regression Model (Page 488)

  • Deterministic Line: y = β₀ + β₁x (β₁ = slope, β₀ = y-intercept).
  • Probabilistic Model: Accounts for uncertainty in y for a given x.
  • Variable Roles:
    • Independent Variable (x): Predictor, explanatory variable. Treated as fixed/given.
    • Dependent Variable (Y): Response variable. Random for a fixed x. We observe value y.
  • Data: n pairs (x₁, y₁), ..., (xn, yn).
  • Scatterplot: Plot of (xᵢ, yᵢ) pairs. First step to visualize the relationship.

  • Scatterplot (Fig 12.1): Shows positive, roughly linear relationship. Not deterministic (multiple y's for same x).

A Linear Probabilistic Model (Page 491)

Definition: The Simple Linear Regression Model (Page 491)

The model equation is:

Y = β₀ + β₁ x + ε   (Equation 12.1)

Where:

  • β₀: True intercept parameter.
  • β₁: True slope parameter.
  • ε: Random error term (deviation).

Assumptions about the error term ε:

  1. E(ε) = 0 (Mean error is zero).
  2. V(ε) = σ² (Variance is constant for all x - homogeneity of variance).
  3. ε is normally distributed.
  4. Errors (ε₁, ..., εn) for different observations are independent.

Implications of the Model Assumptions (Page 492):

  • Mean of Y for a fixed x:
    E(Y | x) = β₀ + β₁ x
    (The mean value of Y lies on the true regression line).
  • Variance of Y for a fixed x:
    V(Y | x) = σ²
    (The variance of Y is constant for all x).
  • Distribution of Y for a fixed x: Y follows a normal distribution N(β₀ + β₁x, σ²).
  • (See Figure 12.4).
  • σ² measures inherent variability around the true line. Small σ² means points are close to the line; large σ² means more scatter. (See Figure 12.9).

Example 12.3: Stress vs. Time-to-Failure (Page 493)

  • Model: Y = 65 − 1.2x + ε, with σ=8 (σ²=64).
  • For x=20, E(Y|20)=41. Y|(x=20) ~ N(41, 64).
  • For x=25, E(Y|25)=35. Y|(x=25) ~ N(35, 64).
  • P(Y > 50 when x=20) = P(Z > (50-41)/8) = P(Z > 1.13) ≈ 0.1292.
  • P(Y > 50 when x=25) = P(Z > (50-35)/8) = P(Z > 1.88) ≈ 0.0301.
  • Comparing Y₁ at x=25 and Y₂ at x=24: P(Y₁ > Y₂) ≈ 0.4562 (significant chance due to variability).

12.2 Estimating Model Parameters (Page 496)

Parameters β₀, β₁, and σ² are unknown. We estimate them from sample data (xᵢ, yᵢ) assuming Yᵢ = β₀ + β₁xᵢ + εᵢ.

Goal: Find estimates β₀ and β₁ for the estimated regression line y = β₀ + β₁x that best fits the data.

Principle of Least Squares (Page 497)

  • Vertical Deviation (Residual): For a candidate line y = b₀ + b₁x, the residual for point (xᵢ, yᵢ) is yᵢ − (b₀ + b₁xᵢ).
  • Sum of Squared Deviations (SSError): Sum the squares of these residuals:
    f(b₀, b₁) = n Σ i=1 [yᵢ − (b₀ + b₁xᵢ)]²
  • Least Squares Principle: Choose b₀ and b₁ to minimize this sum.
  • Least Squares Estimates (β₀, β₁): The values of b₀, b₁ that achieve the minimum.
  • Estimated Regression Line (Least Squares Line): y = β₀ + β₁x.

Finding the Least Squares Estimates (Page 497):

Calculus (minimizing f(b₀, b₁)) leads to the normal equations which are solved to give:

Proposition: Formulas for Least Squares Estimates (Page 498)

Define summary statistics:

  • Sxx = nΣi=1 (xᵢ − x)² = nΣi=1 xᵢ² − (nΣi=1 xᵢ)² n
  • Syy = nΣi=1 (yᵢ − y)² = nΣi=1 yᵢ² − (nΣi=1 yᵢ)² n   (Total Sum of Squares, SST)
  • Sxy = nΣi=1 (xᵢ − x)(yᵢ − y) = nΣi=1 xᵢ yᵢ − (nΣi=1 xᵢ)(nΣi=1 yᵢ) n

The least squares estimate for the slope (β₁) is:

β₁ = Sxy Sxx   (Equation 12.2)

The least squares estimate for the intercept (β₀) is:

β₀ = Σyᵢ − β₁ Σxᵢ n = y − β₁ x   (Equation 12.3)

(Calculate β₁ first, then β₀. Retain precision.)

Example 12.4: Iodine Value and Cetane Number (Page 498)

  • Data: n=14 pairs (x=iodine, y=cetane).
  • Summary Stats given. Calculate Sxx ≈ 6802.77, Sxy ≈ -1424.41 (using text's value).
  • Estimate Slope: β₁ = Sxy / Sxx ≈ -1424.41 / 6802.77 ≈ -0.2094.
  • Estimate Intercept: x ≈ 93.39, y ≈ 55.66. β₀ = y - β₁x ≈ 55.66 - (-0.2094)(93.39) ≈ 75.21.
  • Estimated Regression Line: y = 75.21 - 0.2094x.
  • Interpretation: Estimated average cetane number decreases by 0.2094 for each 1g increase in iodine value.
  • (See Figure 12.8).

Using the Estimated Line (Page 499)

The line y = β₀ + β₁x is used for:

  • Point Estimation: Estimating E(Y | x=x*). Estimate is y = β₀ + β₁x*.
  • Point Prediction: Predicting a single future Y value at x=x*. Prediction is also y = β₀ + β₁x*.

Example 12.5: Point Estimate/Prediction (Page 499)

  • Using y = 75.21 - 0.2094x. For x*=100:
  • Estimate E(Y | x=100): y = 75.21 - 0.2094(100) ≈ 54.27.
  • Predict single Y at x=100: Prediction is also 54.27.
  • Extrapolation Warning: Avoid using the line for x values far outside the observed data range.

Estimating σ² and σ (Page 500)

Estimate variance σ² around the true line based on deviations from the estimated line.

Definition: Fitted Values and Residuals (Page 500)

  • Fitted (Predicted) Values (yᵢ): Values on the estimated line for each xᵢ.
    yᵢ = β₀ + β₁ xᵢ
  • Residuals (eᵢ): Observed yᵢ minus fitted yᵢ.
    eᵢ = yᵢ − yᵢ = yᵢ − (β₀ + β₁ xᵢ)
    (eᵢ is the vertical distance from point i to the estimated line).

Definition: Error Sum of Squares and Estimates of σ² and σ (Page 502)

  • Error Sum of Squares (SSE): Sum of squared residuals. Measures unexplained variation.
    SSE = nΣi=1 eᵢ² = nΣi=1 (yᵢ − yᵢ)²
  • Estimate of σ²:
    s² = SSEn − 2
    (Divisor is n−2 degrees of freedom because we estimated 2 parameters, β₀ and β₁).
  • Estimate of σ:
    s = √s² = √[ SSEn − 2 ]
    (s is the standard error of the estimate, typical size of a residual).
  • Computational Formula for SSE:
    SSE = Syy − β₁ Sxy
    (Watch rounding of β₁).

Example 12.6/12.7: Sludge Moisture Content (Pages 500-502)

  • Data: y=moisture vs. x=filtration rate (n=20).
  • Estimated line: y = 72.96 + 0.041x.
  • SSE from residuals: SSE = Σeᵢ² = 7.968.
  • Estimate of σ²: s² = SSE / (n - 2) = 7.968 / 18 ≈ 0.4427.
  • Estimate of σ: s = √0.4427 ≈ 0.665. (Typical deviation from line ≈ 0.665%).

Example 12.8: Attenuation vs. Fracture Strength (Page 503)

  • Data: n=14 pairs. Calculate Sxx, Sxy, β̂₁, β̂₀.
  • Calculate SSE: Using computational formula (or from residuals). Text gets SSE ≈ 0.2625.
  • Estimate of σ²: s² = SSE / (n - 2) = 0.2625 / 12 ≈ 0.02187.
  • Estimate of σ: s = √0.02187 ≈ 0.1479.

The Coefficient of Determination (r²) (Page 503)

Measures the proportion of variability in y explained by the linear relationship with x.

  • Total Sum of Squares (SST): Total variability in y.
    SST = Syy = nΣi=1 (yᵢ − y)²
  • Error Sum of Squares (SSE): Unexplained variability (around the line).
    SSE = nΣi=1 (yᵢ − yᵢ)²
  • Coefficient of Determination (r²): Proportion of variation explained by regression.
    r² = 1 − SSESST
  • Range: 0 ≤ r² ≤ 1.
  • r² ≈ 1: Good fit, model explains most variation.
  • r² ≈ 0: Poor fit, model explains little variation.
  • Alternative Formula: r² = SSR / SST, where SSR = SST - SSE (Regression Sum of Squares).

Example 12.9: r² for Iodine/Cetane Data (Page 505)

  • SST = Syy ≈ 377.174.
  • SSE ≈ 78.920 (calculated via shortcut).
  • r² = 1 - (78.920 / 377.174) ≈ 1 - 0.209 = 0.791.
  • Interpretation: Approx. 79.1% of variation in cetane number is explained by linear regression on iodine value. Strong relationship.
  • (Minitab Output Figure 12.12 confirms R-sq = 79.1%).

12.5 Correlation (Page 527)

Correlation analysis measures the strength and direction of the linear relationship between two variables (X, Y), treating both as random.

The Sample Correlation Coefficient (r) (Page 528)

Measures linear relationship in the sample (xᵢ, yᵢ).

  • Motivation: Sxy indicates direction, but magnitude depends on units.
  • Definition: Standardize Sxy.
    r = Sxy √Sxx √Syy   (Equation 12.8)

Example 12.15: Corn Yield (x) vs Peanut Yield (y) (Page 528)

  • Data: n=8 pairs. Calculate Sxx=5.75, Syy=0.5124, Sxy=0.5960.
  • Calculate r: r = 0.5960 / (√5.75 * √0.5124) ≈ 0.347.

Properties of r (Page 529)

  • Symmetric in x and y.
  • Unitless.
  • Range: −1 ≤ r ≤ +1.
  • r = +1: Perfect positive linear relationship.
  • r = −1: Perfect negative linear relationship.
  • r²: Literally the square of r. Same as coefficient of determination in regression.
  • Interpretation:
    • r close to ±1: Strong linear relationship.
    • r close to 0: Weak linear relationship. (Could still be a non-linear relationship!).
  • Rule of Thumb for Strength: Weak: |r|<0.5; Moderate: 0.5≤|r|<0.8; Strong: |r|≥0.8.
  • (Example 12.15: r=0.347 is weak positive).

Inferences About the Population Correlation Coefficient (ρ) (Page 530)

  • r estimates the population correlation coefficient ρ (rho).
  • ρ = Cov(X,Y) / (σX σY). Measures linear relationship in the population.
  • Assumption for Inference: Sample is from a bivariate normal distribution (stronger than regression assumptions). If true, ρ=0 implies independence.

Testing for Absence of Correlation (H₀: ρ = 0) (Page 531)

  • Checks for any significant linear relationship.
  • Test Statistic: If H₀: ρ=0 is true (and bivariate normal):
    T = R √[n − 2] √[1 − R²]
    follows a t distribution with n − 2 degrees of freedom. (R is the estimator for ρ, we use sample r).
  • P-value: Calculate t = (r√[n − 2]) / √[1 − r²]. Use tn-2 distribution based on H<0xE2><0x82><0x90> (>0, <0, or ≠0).
  • Equivalence: This test is equivalent to the t-test for H₀: β₁ = 0 in regression.

Example 12.16: Maternal Age and Birth Weight (Page 531)

  • Data: n=10, r = 0.884. Test H₀: ρ=0 vs H<0xE2><0x82><0x90>: ρ > 0.
  • df=8. t = (0.884 * √8) / √(1 - 0.884²) ≈ 5.34.
  • P-value (upper-tailed, df=8) ≈ 0.0003.
  • Decision: Reject H₀. Strong evidence of positive linear relationship.

Example 12.17: Manganese Exposure (Page 532)

  • Control Sample: n=45, r=0.29. Test H₀: ρ=0 vs H<0xE2><0x82><0x90>: ρ ≠ 0. t≈1.99. df=43. P-value ≈ 0.052. Fail to reject H₀ at α=0.05.
  • Exposed Sample: n=22, r=0.83. Test H₀: ρ=0 vs H<0xE2><0x82><0x90>: ρ ≠ 0. t≈6.7. df=20. P-value ≈ 0. Reject H₀.

Other Inferences Concerning ρ (ρ₀ ≠ 0) (Page 532)

Use Fisher's transformation for H₀: ρ = ρ₀ (≠0) or for CI for ρ.

  • Transformation:
    V = 12 ln( 1 + R 1 − R )   (Equation 12.9)
    (R is estimator for ρ).
  • Distribution: V is approx. Normal with:
    • Mean: μV = (1/2) ln[ (1 + ρ) / (1 - ρ) ]
    • Variance: σ²V = 1 / (n − 3)

Test for H₀: ρ = ρ₀ (ρ₀ ≠ 0) (Page 533)

  • Calculate v from sample r. Calculate μV,0 using ρ₀.
  • Test Statistic:
    Z = v − μV,0 1 / √[n − 3]
  • P-value: Use standard normal distribution based on H<0xE2><0x82><0x90>.

Confidence Interval for ρ (Page 534)

  1. Calculate v = (1/2)ln[(1+r)/(1-r)].
  2. Find 100(1-α)% CI for μV:
    v ± zα/2 1 √[n − 3]
    Let the limits be (c₁, c₂).
  3. Transform limits back to ρ scale: CI for ρ is
    ( e2c₁ − 1 e2c₁ + 1 , e2c₂ − 1 e2c₂ + 1 )

Example 12.18: Cube Strength (x) vs Cylinder Strength (y) (Page 533)

  • Data: n=19, r=0.761. Test H₀: ρ=0.5 vs H<0xE2><0x82><0x90>: ρ > 0.5.
  • Calculate v ≈ 0.999. μ<0xE1><0xB5><0x83>,₀ ≈ 0.549. σ<0xE1><0xB5><0x83> = 1/√16 = 0.25.
  • Z = (0.999 - 0.549) / 0.25 = 1.80.
  • P-value (upper-tailed) = 1 - Φ(1.80) ≈ 0.0359.
  • Decision: Reject H₀ at α=0.05. Moderate evidence ρ > 0.5.

Example 12.19: Height (x) vs Wingspan (y) (Page 534)

  • Data: n=16, r=0.9422. Find 95% CI for ρ.
  • Calculate v ≈ 1.757.
  • 95% CI for μ<0xE1><0xB5><0x83>: 1.757 ± 1.96 * (1/√13) ≈ 1.757 ± 0.544 ≈ (1.213, 2.301). Let (c₁, c₂) = (1.213, 2.301).
  • Transform back: Lower limit for ρ ≈ (e^(2*1.213) - 1)/(e^(2*1.213) + 1) ≈ 0.838. Upper limit for ρ ≈ (e^(2*2.301) - 1)/(e^(2*2.301) + 1) ≈ 0.980.
  • 95% CI for ρ is (0.838, 0.980). Suggests strong linear association.
