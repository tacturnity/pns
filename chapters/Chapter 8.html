Chapter 8: Tests of Hypotheses Based on a Single Sample

Introduction (Page 310)

In the previous chapter, we learned how to estimate a population parameter (like μ or p) using either a single number (a point estimate) or an interval of plausible values (a confidence interval).

This chapter focuses on a different aspect of statistical inference called hypothesis testing. Instead of estimating the value of a parameter, our goal here is often to decide between two contradictory claims about the parameter.

For example, we might want to decide if the true average diameter of a pipe is exactly 0.75 inches or if it's different from 0.75 inches. Or we might want to determine if a new process truly reduces the proportion of defective items below 10%.

This chapter will cover:

  • Basic concepts and terminology of hypothesis testing.
  • Decision procedures (tests) for common scenarios involving a sample from a single population.

Section 8.1: Hypotheses and Test Procedures (Page 311)

Statistical Hypothesis: A claim or assertion about:

  • A single parameter (e.g., μ = 75, p < 0.10, σ > 0.001)
  • Several parameters (e.g., μ1 − μ2 = 0)
  • The form of a probability distribution (e.g., the data follows a normal distribution - covered later).

This chapter focuses on hypotheses about single parameters.

Two Contradictory Hypotheses: Every hypothesis testing problem involves choosing between two opposing claims.

Analogy: Criminal Trial:

  • Claim 1: Accused is innocent.
  • Claim 2: Accused is guilty.
  • The claim of innocence is initially favored ("protected"). Requires strong evidence for guilt.

Statistical Hypotheses Formulation: Similarly, one claim is initially favored. We require strong evidence from data to abandon it.

Definition: Null and Alternative Hypotheses (Page 311)

  • Null Hypothesis (H0): The claim initially assumed true. Often represents "no effect," "status quo."
  • Alternative Hypothesis (Ha): The assertion contradicting H0. Often what the researcher wants evidence for.

The Decision Process:

  1. Start assuming H0 is true.
  2. Collect sample data.
  3. Evaluate if data strongly contradicts H0.
  4. Decision:
    • If evidence is strong against H0, reject H0 (in favor of Ha).
    • If evidence is not strong, fail to reject H0 (H0 remains plausible).

Test of Hypotheses: A method using sample data to decide whether H0 should be rejected.

Example: Seeking Evidence for Change (Page 311-312)

  • Company considers new bearing coating. Current average life is 1000 hours. Want evidence it's better.
  • Let μ = true average life with new coating.
  • Null Hypothesis: H0: μ = 1000 (status quo).
  • Alternative Hypothesis: Ha: μ > 1000 (claim needing strong evidence).
  • Only reject H0 if data strongly supports Ha.

H0 as an Equality Claim (Page 312):

  • In this text, H0 usually stated as equality (e.g., H0: θ = θ0), where θ0 is the null value.
  • Even if situation suggests inequality (e.g., H0: p ≥ 0.10), often simplified to H0: p = 0.10 for the test procedure.
  • Rationale: Test designed for the boundary case (θ = θ0) is generally valid for the broader inequality.

Forms of the Alternative Hypothesis (Ha): Given H0: θ = θ0, Ha is typically one of:

  • Ha: θ > θ0 (upper-tailed test)
  • Ha: θ < θ0 (lower-tailed test)
  • Ha: θ ≠ θ0 (two-tailed test)

Test Procedures and P-Values (Page 312)

  • Test Procedure: A rule, based on sample data, for deciding whether to reject H0.
  • Core Question: If H0 is true, how likely is it to observe sample results at least as contradictory to H0 as what we observed?

Example 8.1: Yogurt Taste Test (Page 313)

  • Goal: See if Brand D can win over Brand C consumers. n=100 consumers.
  • Parameter: p = proportion preferring C over D.
  • Hypotheses: H0: p = 0.5 (no preference) vs. Ha: p < 0.5 (majority prefer D).
  • Test Statistic: X = number preferring C. If H0 true, expect X ≈ 50. Small X contradicts H0.
  • Observed Data 1: x = 37 prefer C.
    • Values at least as contradictory: X ≤ 37.
    • Probability (P-value): Assume H0 true, X ~ Binomial(100, 0.5). P(X ≤ 37 | p=0.5) = B(37; 100, 0.5) = 0.006.
    • Interpretation: Very unlikely (0.6% chance) if H0 true. Strong evidence against H0. Reject H0.
  • Observed Data 2: x = 45 prefer C.
    • Values at least as contradictory: X ≤ 45.
    • P-value: P(X ≤ 45 | p=0.5) = B(45; 100, 0.5) = 0.184.
    • Interpretation: Not very unlikely (18.4% chance) if H0 true. Not strong evidence against H0. Fail to reject H0.

Example 8.2: Freshman Weight Gain (Pages 314-315)

  • Goal: Test belief that average gain μ = 15 lbs.
  • Hypotheses: H0: μ = 15 vs. Ha: μ ≠ 15.
  • Test Statistic (Large Sample, n > 40): Assume H0 true.
    Z = X − μ0 S/√n = X − 15 S/√n
    Z is approx. N(0,1) when H0 true.
  • Observed Data 1: Suppose n=137, x=13.7, calculated z = -2.80.
    • Values at least as contradictory (two-tailed): Z ≤ -2.80 OR Z ≥ +2.80.
    • P-value: Assume H0 true, Z ~ N(0, 1). P(Z ≤ -2.80 or Z ≥ +2.80) = 2 * P(Z ≥ 2.80) = 2 * [1 - Φ(2.80)] = 2 * [1 - 0.9974] = 0.0052.
    • Interpretation: Very unlikely (0.52% chance) if H0 true. Strong evidence against H0. Reject H0.
  • Actual Study Data: n=137, x = 2.42 lbs, s = 5.72 lbs. Calculated z = -25.7. P-value ≈ 0. Overwhelming evidence against H0.

Definitions: Test Statistic, P-value, Significance Level (Page 316)

  • Test Statistic: Function of sample data used to decide about H0.
  • P-value: Probability, assuming H0 is true, of obtaining a test statistic value at least as contradictory to H0 as the observed value.
    • Smaller P-value ⇒ Stronger evidence against H0.
  • Significance Level (α): Threshold chosen beforehand (e.g., 0.05, 0.01) for decision making.
  • Decision Rule:
    • If P-value ≤ α, reject H0.
    • If P-value > α, fail to reject H0.
  • Common α levels: 0.05, 0.01, 0.001, 0.10. Smaller α protects H0 more.

Important Points about P-value:

  • It is NOT P(H0 is true) or P(H0 is false).
  • It is NOT P(making a wrong decision).

Example 8.3: Battery Zinc Mass (Page 317)

  • Goal: Determine if true average zinc mass (μ) exceeds 2.0 g.
  • Hypotheses: H0: μ = 2.0 vs. Ha: μ > 2.0.
  • Data: n = 51 (large), x = 2.06 g, s = 0.141 g. α = 0.01.
  • Test Statistic: Z = (X - 2.0) / (S/√51). Calculated z = 3.04.
  • P-value (upper-tailed): P = P(Z ≥ 3.04 | H0 true) = 1 - Φ(3.04) = 1 - 0.9988 = 0.0012.
  • Decision: P-value (0.0012) ≤ α (0.01). Reject H0.
  • Conclusion: Compelling evidence (at α=0.01) that true average zinc mass exceeds 2.0 g.

Errors in Hypothesis Testing (Page 317)

Two possible errors:

  • Type I Error: Rejecting H0 when H0 is true. (Convicting the innocent).
  • Type II Error: Failing to reject H0 when H0 is false (i.e., Ha is true). (Letting the guilty go free).

Probabilities of Errors: α and β

  • α (Alpha): Probability of Type I error. α = P(Reject H0 | H0 true). This is the significance level.
  • β (Beta): Probability of Type II error. β = P(Fail to reject H0 | H0 false).
    • β depends on the specific value of the parameter within Ha.

Example 8.4: Bumper Crashes (Pages 318-319)

  • Test H0: p = 0.25 vs Ha: p > 0.25. n = 20. Test statistic X = number no-damage.
  • Decision Rule (approx. α=0.10): Reject H0 if X ≥ 8. (Actual α = P(X ≥ 8 | p=0.25) = 0.102).
  • Calculating β: Depends on true p' > 0.25. Fail to reject if X ≤ 7.
    • If p' = 0.30: β(0.30) = P(X ≤ 7 | p=0.30) = B(7; 20, 0.30) = 0.772 (High).
    • If p' = 0.40: β(0.40) = P(X ≤ 7 | p=0.40) = B(7; 20, 0.40) = 0.416.
    • If p' = 0.50: β(0.50) = P(X ≤ 7 | p=0.50) = B(7; 20, 0.50) = 0.132.
  • Observation: β decreases as true p' gets further from null p0. High β here due to small n.

Example 8.5: Paint Drying Time (Pages 319-321)

  • Test H0: μ = 75 vs Ha: μ < 75. Normal pop, σ=9, n=25.
  • Test Statistic: Z = (X - 75) / (9/√25) = (X - 75) / 1.8.
  • Decision Rule (α=0.01): Reject H0 if calculated z ≤ -2.33. (Equivalent to rejecting if X ≤ 70.8). Fail to reject if X > 70.8.
  • Calculating α: α = P(Z ≤ -2.33 | μ=75) = 0.01.
  • Calculating β: Depends on true μ' < 75.
    • If μ' = 72: β(72) = P(Fail H0 | μ=72) = P(X > 70.8 | μ=72) = P(Z > (70.8-72)/1.8) = P(Z > -0.67) = 1 - Φ(-0.67) = 0.7486.
    • If μ' = 70: β(70) = P(X > 70.8 | μ=70) = P(Z > (70.8-70)/1.8) = P(Z > 0.44) = 1 - Φ(0.44) = 0.3300.
    • If μ' = 67: β(67) = P(X > 70.8 | μ=67) = P(Z > (70.8-67)/1.8) = P(Z > 2.11) = 1 - Φ(2.11) = 0.0174.
  • Observation: β decreases as true μ' moves further below null μ0. (See Figure 8.3).

Relationship Between α and β (Page 322)

  • Proposition: For fixed n, decreasing α increases β.
  • Trade-off: Can't make both arbitrarily small without increasing n.
  • Practical Approach: Choose largest tolerable α (e.g., 0.05) to minimize β. If β still too high, increase n.

Proposition: Significance Level α = P(Type I error) (Page 322)

The test procedure "Reject H0 if P-value ≤ α" has P(Type I error) = α.

Some Further Comments on the P-Value (Page 323)

  • Observed Significance Level (OSL): P-value is the smallest α at which H0 would be rejected.
  • Reporting: Reporting the P-value is more informative than just Reject/Fail to Reject. Allows readers to use their own α.

Section 8.2: z Tests for Hypotheses about a Population Mean

Case 1: Normal Population Distribution with Known σ (Page 327)

  • Assumptions: Population normal, σ known.
  • Hypotheses: H0: μ = μ0 vs. Ha (> or < or ≠ μ0).
  • Test Statistic: Z = (X - μ0) / (σ/√n). Follows N(0,1) if H0 true.
  • P-value Calculation (using calculated z from sample x):
    • Ha: μ > μ0: P = 1 - Φ(z)
    • Ha: μ < μ0: P = Φ(z)
    • Ha: μ ≠ μ0: P = 2 * [1 - Φ(|z|)]
  • Decision: Reject H0 if P-value ≤ α.

Recommended Steps for Hypothesis Testing (Page 329):

  1. Identify parameter (μ).
  2. State H0 (μ = μ0).
  3. State Ha (> or < or ≠).
  4. Give test statistic formula (Z).
  5. Compute sample quantities and calculate test statistic value (z).
  6. Determine P-value.
  7. Compare P-value to α and state conclusion.

Example 8.6: Sprinkler Activation Temperature (Page 329)

  • H0: μ = 130°F vs. Ha: μ ≠ 130°F. Assume normal, σ=1.5°F.
  • Data: n=9, x=131.08°F.
  • z = (131.08 - 130) / (1.5 / √9) = 2.16.
  • P-value (two-tailed): P = 2 * [1 - Φ(2.16)] = 0.0308.
  • Decision (α = 0.01): P-value (0.0308) > α (0.01). Fail to reject H0.
  • Conclusion: At α=0.01, data does not provide strong evidence that true average differs from 130°F.

β and Sample Size Determination (σ known) (Pages 330-331)

  • Formulas exist for β (Type II error prob.) for a specific alternative μ'.
    • For Ha: μ > μ0:
      β(μ') = Φ( zα + μ0 − μ' σ/√n )
    • Similar formulas for lower and two-tailed tests.
  • Formulas exist for n needed for specified α and β(μ').
    • One-tailed:
      n = [ σ(zα + zβ) μ0 − μ' ]2
    • Two-tailed (approx):
      n ≈ [ σ(zα/2 + zβ) μ0 − μ' ]2
    • (Here zβ cuts off area β in upper tail, e.g., z.10=1.28).

Example 8.7: Tire Tread Life (Page 331)

  • H0: μ = 30000 vs Ha: μ > 30000. Normal, σ=1500.
  • Find β if μ'=31000, n=16, α=0.01 (z.01=2.33). β(31000) = Φ(2.33 + (30000 - 31000) / (1500/√16)) = Φ(-0.34) = 0.3669.
  • Find n for α=0.01, β(31000)=0.10 (z.10=1.28). n = [ 1500(2.33 + 1.28) / (30000 - 31000) ]² = 29.32. Need n = 30.

Case 2: Large-Sample Tests (σ Unknown) (Pages 331-332)

  • Assumptions: n large (n>40), σ unknown. No specific population distribution shape needed.
  • Test Statistic: Replace σ with sample S.
    Z = X − μ0 S/√n
    Z is approx. N(0,1) if H0 true.
  • Procedure: Same as Case 1, using S to calculate z. Conclusions are approximate.

Example 8.8: Dynamic Cone Penetrometer (DCP) (Pages 332-333)

  • H0: μ = 30 vs Ha: μ < 30.
  • Data: n=52 (large), x=28.76, s=12.2647.
  • z = (28.76 - 30) / (12.2647 / √52) = -0.73.
  • P-value (lower-tailed): P = Φ(-0.73) = 0.2327.
  • Decision (α = 0.05): P-value (0.2327) > α (0.05). Fail to reject H0.
  • Conclusion: Not compelling evidence that true average DCP is less than 30.

Section 8.3: The One-Sample t Test (Page 335)

Applies when n is small, σ is unknown, but population is assumed normal.

  • Motivation: Z statistic using S is not N(0,1) for small n.
  • Test Statistic:
    T = X − μ0 S/√n
  • Distribution when H0 is true: T follows a t distribution with n-1 degrees of freedom (df).
  • Procedure: Use t-distribution for P-values.
  • P-value Calculation (using calculated t from sample):
    • Ha: μ > μ0: P = P(T ≥ t) (Area right of t under tn-1 curve)
    • Ha: μ < μ0: P = P(T ≤ t) (Area left of t under tn-1 curve)
    • Ha: μ ≠ μ0: P = 2 * P(T ≥ |t|) (Twice the area in tail beyond |t|)
  • Use t-tables (Appendix Table A.8) or software for P-values.

Example 8.9: Carbon Nanofiber Failure Stress (Page 337)

  • H0: μ = 500 MPa vs. Ha: μ > 500 MPa. Assume normal pop.
  • Data: n=19, x=562.68, s=180.874.
  • t = (562.68 - 500) / (180.874 / √19) = 1.51. df=18.
  • P-value (upper-tailed): P = P(T ≥ 1.51 | df=18). From Table A.8, 0.063 < P < 0.075. (Minitab: P=0.074).
  • Decision (α = 0.05): P-value (≈0.074) > α (0.05). Fail to reject H0.
  • Conclusion: Insufficient evidence at α=0.05 to conclude true average stress > 500 MPa.

Example 8.10: Time Perception After Smoking Abstinence (Page 338)

  • H0: μ = 45 sec vs. Ha: μ ≠ 45 sec. Assume normal pop.
  • Data: n=20, x=59.30, s=9.84.
  • t = (59.30 - 45) / (9.84 / √20) = 6.50. df=19.
  • P-value (two-tailed): P = 2 * P(T ≥ 6.50 | df=19) ≈ 0.
  • Decision (α = 0.05): P-value (≈0) ≤ α (0.05). Reject H0.
  • Conclusion: Very strong evidence that true average perceived time differs from 45 sec.

β and Sample Size Determination for t-Tests (Pages 338-340)

  • Calculating β for t-tests is complex (involves non-central t-distribution).
  • Use Operating Characteristic (OC) Curves (Appendix Table A.17) or software.
    • OC curves plot β vs. d = |μ0 - μ'| / σ (requires estimate of σ).
  • Power = 1 - β (probability of correctly rejecting a false H0). Software often works with power.
  • Can use OC curves or software to find n for desired α, β, μ', and estimated σ.

Example 8.11: Voltage Drop Test Power (Pages 340-341)

  • H0: μ = 2.5 vs Ha: μ > 2.5. α=0.05. Guess σ=0.100.
  • Find β if μ' = 2.6, n=10 (df=9). d = |2.5-2.6|/0.1 = 1.0. From OC curve, β ≈ 0.1.
  • Find n for β=0.05 at μ'=2.6 (d=1.0). From OC curve, need df≈14, so n=15. (Minitab gives n=13).

Variation in P-values (Page 341)

  • P-value is a random variable.
  • If H0 true, P-value distribution is Uniform(0, 1).
  • If H0 false, P-value distribution concentrated near 0.
  • P-values near α may not be reproducible.

Section 8.4: Tests Concerning a Population Proportion (Page 346)

Testing hypotheses about population proportion p.

Large-Sample Tests (Page 346)

  • Assumptions: n large (check np0 ≥ 10 and n(1-p0) ≥ 10).
  • Hypotheses: H0: p = p0 vs. Ha (> or < or ≠ p0).
  • Rationale: p ≈ Normal. If H0 true, E(p)=p0, StdDev = σ0 = √[p0(1-p0)/n].
  • Test Statistic: Standardize p using null value p0 in std dev:
    Z = p − p0 √[p0(1−p0)/n]
    Z is approx. N(0,1) if H0 true.
  • P-value Calculation (using calculated z from sample p):
    • Ha: p > p0: P = 1 - Φ(z)
    • Ha: p < p0: P = Φ(z)
    • Ha: p ≠ p0: P = 2 * [1 - Φ(|z|)]
  • Decision: Reject H0 if P-value ≤ α.

Example 8.13: Texting During Exams (Page 348)

  • H0: p = 0.05 vs. Ha: p > 0.05.
  • Data: n=267, x=27. p = 0.1011. Validity checks ok.
  • z = (0.1011 - 0.05) / √[0.05(0.95)/267] = 3.84.
  • P-value (upper-tailed): P = 1 - Φ(3.84) ≈ 0.00006.
  • Decision (α = 0.001): P-value (≈0.00006) ≤ α (0.001). Reject H0.
  • Conclusion: Very strong evidence proportion > 5%.

β and Sample Size Determination for z Test for p (Pages 348-349)

  • Formulas exist for β(p') (similar to z-test for μ, but more complex standard deviation under Ha).
  • Formulas exist for n for specified α, β, p₀, p'.
    • One-tailed:
      n = [ zα√p0(1−p0) + zβ√p'(1−p') p' − p0 ]2
    • Two-tailed (approx):
      n ≈ [ zα/2√p0(1−p0) + zβ√p'(1−p') p' − p0 ]2

Example 8.14: Package Delivery Service (Page 349)

  • H0: p = 0.9 vs Ha: p < 0.9. α=0.01. n=225.
  • Find β if true p' = 0.8. Using formula, β(0.8) ≈ 0.0228.
  • Find n for α=0.01, β(0.8)=0.01 (z.01=2.33). n = [ (2.33√0.9(0.1) + 2.33√0.8(0.2)) / (0.8 - 0.9) ]² ≈ 266.

Small-Sample Tests (Page 349)

  • Use when large-sample conditions fail (np0<10 or n(1-p0)<10).
  • Use the Binomial distribution directly.
  • Test Statistic: X = number of successes.
  • P-value Calculation (using Binomial probabilities):
    • Ha: p > p0: P = P(X ≥ x | p=p0)
    • Ha: p < p0: P = P(X ≤ x | p=p0)
    • Ha: p ≠ p0: P = 2 * min[ P(X ≤ x), P(X ≥ x) ] (calculated at p=p0)
  • Achieving α: Due to discrete nature, choose rejection rule giving α closest to target without exceeding it.

Example 8.15: Plastic Trash Can Warranty (Page 350)

  • H0: p = 0.9 vs Ha: p < 0.9.
  • Data: n=20. Test statistic X = number surviving.
  • Achieving α ≈ 0.05: From Binomial(20, 0.9) table: P(X ≤ 15) = B(15; 20, 0.9) = 0.043. P(X ≤ 16) = B(16; 20, 0.9) = 0.133. Choose rule: Reject H0 if X ≤ 15 (gives α=0.043).
  • Calculating β: If true p' = 0.8. Fail to reject if X ≥ 16. β(0.8) = P(X ≥ 16 | p=0.8) = 1 - B(15; 20, 0.8) = 1 - 0.370 = 0.630. (High).
